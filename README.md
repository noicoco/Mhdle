# MHdle - A Monster Hunter™ 'Wordle-like' WebApp

## What is it ?
CS Student project driven by our many duo hunts on Wilds. 🐉⚔️

Putting our many hours in class to good use. Using Vue.js

## Features
- Accurate-ish data
- Multi Platform
- Different game modes
  - Guess The Monster
  - Monster Cry
  - Guess the Talent
  - Compendium Knowledge Check

Most features as of time of writing this are in progress. `(27/03/2026)`

The data is fine though, no worries.

## Goals
Currently, the project only features MH Wilds' monsters and general data

Our goal is to make it span across the entire Monster Hunter saga.
